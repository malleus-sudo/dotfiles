# My dotfiles

## Description
Ricing for personal use. Thematic from my RPG. Gothic + Cyberpunk + Arcane style. Feel free to use and adapt as you wish. I intend to keep it a simple and functional rice, but still pretty someway.

## Tools
OS: Archlinux
WM: Hyprland
IDE: vscode

## Install
Dependencies:

```bash
sudo pacman -S hyprland hyprpaper waybar rofi hyprcursor
```

For wlogout

```bash
yay -S wlogout
```
